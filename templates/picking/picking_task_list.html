Задачи подбора

Задачи подбора создаются автоматически при подтверждении заказа. Каждая задача соответствует определённой зоне склада (ячейки, полки, напольное хранение).

Фильтрация

  • Статус — фильтр по статусу задачи (ожидают, в работе, завершены)
  • Тип зоны — фильтр по типу зоны (ячейки, полки, напольное)

Выполнение подбора

Откройте задачу подбора для просмотра деталей. Сканируйте товары и подтверждайте подбор. После завершения всех задач подбора заказ переходит в статус "Готов к отгрузке".

Статусы

  • Ожидают подбора — задача создана, но ещё не начата
  • В работе — подбор начат
  • Завершена — все товары подобраны

Автоматизация

При завершении задачи подбора автоматически обновляются остатки на складе и статус заказа.
