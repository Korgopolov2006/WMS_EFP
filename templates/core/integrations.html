Интеграции и API

Архитектура обмена с 1С:Розница, внешними каталогами и онлайн-витриной построена вокруг стабильного REST API.

API v1: /api/v1/... Bearer токены для сервисов

Каналы интеграции

Каждый внешний сервис подключается как отдельный канал с токеном роли INTEGRATION.

1С:Розница
Синхронизация номенклатуры, цен и остатков через JSON API.
TecDoc / каталоги
Обогащение карточек товара и подбор по VIN/OEM.
Онлайн-витрина
Публикация остатков и резервирование заказов.

1С:Розница

Обмен справочниками и документами продаж.

Номенклатура: GET /api/v1/products/
Остатки/резервы: планируется модуль хранения
Документы продаж: планируется модуль отгрузки

Онлайн-витрина

Витрина работает поверх тех же REST-эндпоинтов, что и 1С.

Поиск по OEM/аналогу: GET /api/v1/products/?oem=...
Карточка товара: GET /api/v1/products/<id>/

Сервисы подбора

Внешние каталоги могут использовать API для сопоставления OEM ↔ аналоги.

Перекрёстные ссылки: GET /api/v1/products/<id>/xrefs/
Приёмка с авторазмещением: POST /api/v1/receivings/<id>/scan/
