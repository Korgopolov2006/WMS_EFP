Создание приёмки

Нажмите кнопку + Новая приёмка для создания нового документа приёмки товара.

Статусы приёмки

  • Черновик — документ создан, но не завершён
  • В процессе — приёмка начата
  • Завершена — приёмка завершена, остатки созданы на складе

Фильтрация

Используйте фильтр по статусу для поиска нужных документов приёмки.

Завершение приёмки

После завершения приёмки автоматически создаются остатки на складе для каждой позиции. Также автоматически выполняются backorder для товаров, которых было недостаточно.

Просмотр деталей

Нажмите кнопку Открыть для просмотра деталей приёмки, добавления строк и завершения документа.
