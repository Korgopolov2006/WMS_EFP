Создание инвентаризации

Нажмите кнопку + Новая инвентаризация для создания нового документа инвентаризации.

Статусы инвентаризации

  • Черновик — документ создан, но не начат
  • В процессе — инвентаризация начата, идёт подсчёт остатков
  • Завершена — инвентаризация завершена, остатки обновлены

Выполнение инвентаризации

После начала инвентаризации заполните фактические количества для каждой позиции. После завершения инвентаризации остатки на складе автоматически обновляются по фактическим данным.

Фильтрация

Используйте фильтр по статусу для поиска нужных документов инвентаризации.

Просмотр деталей

Нажмите кнопку Открыть для просмотра деталей инвентаризации, заполнения фактических количеств и завершения документа.
