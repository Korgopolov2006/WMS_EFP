Фильтрация задач

Используйте фильтры для поиска нужных задач:

  • Тип задачи — фильтр по типу (приёмка, инвентаризация, подбор, отгрузка)
  • Статус — фильтр по статусу (ожидают, в работе, завершены)
  • Только мои — показывать только назначенные вам задачи (для администраторов)

Выполнение задач

Для задач в статусе "Ожидают выполнения" доступна кнопка Начать. После начала задача переходит в статус "В работе".

Для задач в статусе "В работе" доступна кнопка Завершить. После завершения задача закрывается.

Связанные документы

Каждая задача может быть связана с документом (приёмка, инвентаризация, заказ, задача подбора). Переходите по ссылкам для просмотра деталей.

Прогресс выполнения

Прогресс-бары показывают процент выполнения задачи: 0% для ожидающих, 50% для в работе, 100% для завершённых.
